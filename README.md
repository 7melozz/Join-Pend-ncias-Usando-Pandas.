# Join-Pendências Usando Pandas

Este projeto foi desenvolvido para fins profissionais, utilizando a biblioteca Pandas em Python para realizar operações de JOIN entre duas planilhas. O objetivo é aplicar filtros, verificar status e gerar métricas simples para análise na área de atuação.

##Tecnologias utilizadas
- Python
- Pandas
- Jupyter Notebook (opcional)
- Excel / CSV

##Funcionalidades
- Leitura de duas planilhas
- JOIN com base em colunas-chave
- Filtros por status
- Geração de métricas simples

##Objetivo
Automatizar tarefas de análise de pendências entre planilhas, facilitando a tomada de decisão.
